# CONSULT DOCTORThis application platform aims to foster various health related issues/queries of a patient through consultation with a doctor.Proposed Idea:1. Resolves issues/queries of a patient via doctor consultation.2. Saves good time of patient.3. Patient can also take appointment from a doctor.# 24x7 Availability4. Patients can access their lab reports anytime anywhere and on any smart device.# Secured5. Each patient report is password protected which ensures high end to end security and takes care of patient privacy too.# A Cost-Effective Solution6. Patient don’t need to visit the hospital for issues/queries and even for lab reports.7. This app will have additional Features of informing about new emerging and some common diseases with their preventive methods and some home remedies.
